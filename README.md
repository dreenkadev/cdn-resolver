# Cdn Resolver

CDN origin IP resolver.

Resolver IP origin CDN.


## Description

Identifies origin server IP addresses behind CDN services.

## Deskripsi

Mengidentifikasi alamat IP server origin di balik layanan CDN.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Resolve: python cdn_resolver.py example.com
- Demo: python cdn_resolver.py --demo

## Penggunaan

- Resolve: python cdn_resolver.py example.com
- Demo: python cdn_resolver.py --demo
